\title{Mutually Algebraic Sets}

\date{May 2, 2024}

\author{Garrett Peters}

\begin{document}
\maketitle
\par{Let \(X\) be a non-empty set and \(l \geq1\). Then a subset \(C \subseteq  X^l\) is mutually algebraic if there exists some \(K\) such that for all \(a \in  X\), we have \(| \{ \overline {c} \in  C: a \in \overline {c} \} | \leq  K\).}
