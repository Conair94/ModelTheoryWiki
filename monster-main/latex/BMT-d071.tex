\par{Let \(I\) be any set. Then a filter \(\mathcal {F}\) on the power set \(\mathcal {P}(I)\) is a collection of subsets of \(I\) with the following properties:}\par{1: \(I \in \mathcal {F}\).}\par{2: If \(A \in   \mathcal {F}\) and \(A \subseteq  B\), then \(B \in   \mathcal {F}\).}\par{3: If \(A,B  \in   \mathcal {F}\), then we have \(A \cap  B \in   \mathcal {F}\).}\par{Furthermore, a filter is proper if \(\emptyset \notin \mathcal {F}\).}
