\par{Given a language \(\mathcal {L}\), we define a \emph{term} to be any of the following:}\par{If \(c \in   \mathcal {L}\) is a constant symbol, then \(c\) is a term}\par{If \(x\) is a variable symbol, then \(x\) is a term. We generally assume we have countably infinitely many variable symbols}\par{If \(t_1,..., t_n\) are terms, and \(f \in   \mathcal {L}\) is an \(n\)-ary function, then \(f(t_1,..., t_n)\) is a term.}
