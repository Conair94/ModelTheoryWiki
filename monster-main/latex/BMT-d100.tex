\par{
    Given a \ForesterRef{BMT-d001}{language} \(\mathcal {L}\), a \emph{sequent} in \(\mathcal {L}\) is an ordered pair \(( \Gamma , A)\),
    where \(\Gamma\) is a set of \ForesterRef{BMT-d013}{\(\mathcal {L}\)-sentences} and \(A\) is an \ForesterRef{BMT-d009}{\(\mathcal {L}\)-formula}.
    This is usually written \(\Gamma \vdash  A\).
}
