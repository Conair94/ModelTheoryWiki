\par{Given a language \(\mathcal {L}\), an \emph{atomic formula} is defined as follows:}\par{If \(R \in   \mathcal {L}\) is an n-ary relation symbol, and \(t_1, \dots , t_n\) are \ForesterRef{BMT-d007}{terms}, then \(R(t_1, \dots , t_n)\) is an atomic formula.}\par{If \(t_1\) and \(t_2\) are \(\mathcal {L}\)-terms, then \(t_1 = t_2\) is an atomic formula. }
