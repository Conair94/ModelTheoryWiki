\par{We use \emph{quantifiers} to bound variables in a formula in first order logic.}\par{If \(\phi (x)\) is a \ForesterRef{BMT-d009}{formula}, then \(\exists  x ( \phi (x))\) should be translated as "there is an interpretation of \(x\) such that \(\phi (x)\) holds.}\par{If \(\phi (x)\) is a \ForesterRef{BMT-d009}{formula}, then \(\forall  x ( \phi (x))\) should be translated as "for all interpretations of \(x\), \(\phi (x)\) holds.}
