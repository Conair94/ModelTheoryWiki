\par{
    Given a language \(\mathcal  L\), a \emph{partial type} is a \ForesterRef{BMT-d023}{finitely satisfiable} \(mathcal L\)-theory.
}\par{
    A partial type \(p\) is a \emph{complete type} when for every \(\mathcal  L\)-formula \(\phi\), either \(\phi\) is in \(p\)
    or \(\neg \phi\) is in \(p\).
