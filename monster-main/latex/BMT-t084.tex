\par{
    The following are equivalent, given a model \(M\) and a cardinal \(\lambda\).
}\par{
    1: \(M\) is \(\lambda\)-saturated.
}\par{
    2: \(M\) is \(\lambda\)-universal and \(\lambda\)-homogeneous.
}\par{
    3: \(M\) is \(< \aleph _0\)-universal and \(\lambda\)-homogeneous.
}
